Student Technology Requirements and Technical Skills

The following is a list of general technologies and skills that are required and will be used throughout this course. For any additional technologies that may be used or required for this course, please consult the Course Overview and Schedule.

Technology Requirements

You will need access to the following technologies which will be used throughout this course:

  • a laptop or desktop computer
  • Internet access
  • Internet browser (Google Chrome, Mozilla Firefox, Microsoft Edge, or Safari)
  • Word Processing software (Microsoft Word or Google Docs)

Optional: Speakers or headphones to listen to videos. 

Technical Skills

The following technical skills are required and will be used throughout this course:

  • Use the learning management system.
  • Use of email with attachments.
  • Create and submit files in commonly used word processing formats.
